Merge Two Sorted Lists - Leetcode
https://leetcode.com/problems/merge-two-sorted-lists/description/
Intuition
Navigate through both lists, taking the values in sorted order.
Approach
Use a temporary node to navigate through the two lists in a while loop.
Complexity
Time complexity: O(n) as we loop for at most twice the length of the shorter list.
Space complexity: O(1)
Code
Python
Copy
# Definition for singly-linked list. # class ListNode: # def __init__(self, val=0, next=None): # self.val = val # self.next = next class Solution: def mergeTwoLists(self, list1: Optional[ListNode], list2: Optional[ListNode]) -> Optional[ListNode]: if not list1: return list2 if not list2: return list1 if list1.val < list2.val: list3 = list1 list1 = list1.next else: list3 = list2 list2 = list2.next current = list3 while list1 and list2: if list1.val < list2.val: current.next = list1 list1 = list1.next else: current.next = list2 list2 = list2.next current = current.next if list1: current.next = list1 if list2: current.next = list2 return list3
